# ROCK-PAPER-SCISSOR
IT'S JUST A GAME
✊✋✌️ Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript where you play against the computer.
The game includes real-time results, score tracking, and responsive design.

---

🚀 Features

- 🎮 Play Rock, Paper, or Scissors against computer
- 🤖 Random computer choice logic
- 📊 Live score tracking (Player vs Computer)
- 🖥️ Responsive and clean UI
- ⚡ Instant result display
- 🔁 Restart game option

---

🛠️ Tech Stack

- HTML5 – Structure
- CSS3 – Styling & layout
- JavaScript (ES6) – Game logic

---

▶️ How to Run the Project


1. Open the project folder

2. Run the game
   Open "indexone.html" in your browser

---

🎯 Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice → Draw
